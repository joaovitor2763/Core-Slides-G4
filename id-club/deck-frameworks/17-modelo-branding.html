Framework · gestão de marca

Identidade emitida, conhecimento percebido.

A marca opera num circuito: você projeta identidade pela comunicação, o mercado devolve percepção pelo tracking.

Do lado emissor, os atributos que você controla — cultura, personalidade, símbolos, nome. Do lado receptor, o que se forma na cabeça do cliente: consciência e imagem. A gestão de marca é manter os dois lados girando alinhados.
